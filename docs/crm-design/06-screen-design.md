# 06. 画面設計書

## 共通レイアウト

### 管理画面ヘッダー

- 左: ロゴ、システム名
- 中央: ダッシュボード、活動、記録、素材、レビュー
- 右: ユーザー名、権限、ログアウト

### 共通UI

- 一覧はテーブル表示とカード表示を切り替え可能
- 検索、ステータス、種別、担当者、公開日のフィルタを持つ
- 作成・編集フォームは右側にプレビューを持つ
- 保存、レビュー依頼、公開、差し戻しは常に画面右上に固定

## ADM-002 ダッシュボード

### 表示項目

- 下書き件数
- レビュー待ち件数
- 公開予定件数
- 直近公開済み記録
- 最近の操作ログ

### 操作

- 活動を作成
- 記録を作成
- 素材を追加
- レビュー一覧へ移動

## ADM-003 活動一覧

### 表示項目

| 項目 | 内容 |
|---|---|
| タイトル | 活動名 |
| 種別 | 取材、問いの実践、社会課題PJ、記録化 |
| 状態 | 下書き、レビュー中、承認済み、公開済み |
| 担当者 | 作成者または主担当 |
| 更新日 | 最終更新日 |

### 操作

- 新規作成
- 編集
- 複製
- アーカイブ
- 記録を作成

## ADM-005 活動編集

### 入力項目

- タイトル
- slug
- 種別
- 概要
- 本文メモ
- 活動日
- 場所
- 主担当
- 関連素材
- 関連記録
- 公開可否
- 内部メモ

### バリデーション

- タイトル必須
- slugは半角英数字とハイフン
- 公開する場合は概要、アイキャッチ、画像クレジット必須

## ADM-008 記録編集

### 入力項目

- タイトル
- slug
- 種別
- 概要
- 本文
- アイキャッチ
- 本文中画像
- 関連活動
- 公開日
- SEOタイトル
- meta description
- 公開ステータス

### プレビュー

- 公開サイトと同じ幅で本文を表示
- 未公開素材は警告表示
- 大学公式誤認リスクのある文言はレビュー時にチェック対象

## ADM-009 素材一覧

### 表示項目

- サムネイル
- ファイル名
- 種別
- クレジット
- 利用範囲
- 公開可否
- 紐付け先

## ADM-012 レビュー詳細

### 表示項目

- 対象コンテンツ
- 差分
- レビューコメント
- チェックリスト
- 承認履歴

### チェックリスト

- 個人情報が含まれていない
- 未公開情報が含まれていない
- 大学公式団体と誤認されない
- 画像・資料の利用条件を満たす
- 公開サイトの文体に合っている

## ADM-014 ユーザー管理

### 表示項目

- 名前
- メール
- ロール
- 状態
- 最終ログイン

### 操作

- 招待
- ロール変更
- 無効化
- 再招待
